This project focuses on analyzing Ehsan platform data for the year 2024 through the development of an interactive dashboard using Power BI, with the aim of providing an accurate overview of donation and campaign performance across different quarters. The analysis is based on real data, including total donations, number of campaigns created, number of operations, as well as the number and value of completed projects.

The dashboard examines the distribution of total donations across quarterly periods, enabling the identification of changes in donation volume over time. It also presents the number and value of completed projects to assess the platform’s outputs and their relationship with donation levels. In addition, campaign performance is analyzed by classifying campaigns into those that exceeded the target, met the target, or fell below it, providing a clear understanding of campaign success levels.

The project also includes an indicator to measure the achievement rate of campaign targets, reflecting the proportion of campaigns that met or exceeded their goals relative to the total number of campaigns created. This supports the evaluation of overall performance efficiency. These indicators are visualized using various chart types, including column charts for comparing values across quarters, donut charts for illustrating campaign performance distribution, and line charts for highlighting trends over time.

Data preprocessing and transformation were conducted using Power Query, while DAX measures were utilized to develop the required analytical calculations. The dashboard design emphasizes structured and clear information presentation, enabling quick interpretation of key metrics and supporting the monitoring of performance and identification of gaps between actual results and targets over time.
